# TSTool / Command Reference / Overview #

This reference section of the documentation provides information about TSTool commands specific to Campbell Cloud web services.

The TSTool core product also has many commands (see the
[TSTool documentation](https://opencdss.state.co.us/tstool/latest/doc-user/command-ref/overview/)).
Additional functionality may be added in the future in additional commands or by enhancing the core TSTool software.

**<p style="text-align: center;">
Campbell Cloud Plugin TSTool Commands
</p>**

| **Command** | **Description** |
| -- | -- |
| [`ReadCampbellCloud`](ReadCampbellCloud/ReadCampbellCloud.md) | Read one or more time series from Campbell Cloud web services. |
| [TSID for Campbell Cloud](TSID/TSID.md) | Time series identifier command for Campbell Cloud. |
